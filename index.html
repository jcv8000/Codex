Save Location

Changing your save location will immediately restart the application.
Make sure it has its own folder to separate the files from others.

You will have to copy/replace the save.json file and notes folder from the old location to the new location to see your notes.
This can be used for cloud syncing your notes, check here for a tutorial.
Unused Files

When you delete a notebook/page in Codex, the actual files that hold the pages' content stays in the notes folder in case you need to get it back.

Don't worry about this unless you have extremely large notes with embedded images and you need extra space. Most note files are less than 1 MB in size.

This button deletes all files that are currently unused by your notebooks.
